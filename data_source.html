Data Source

Our analysis uses the BMW Sales Dataset (2010-2024), obtained from Kaggle.

Dataset Link: BMW Sales Dataset on Kaggle

Dataset Overview

Size and Scope

  • Total records: ~50,000 sales transactions
  • Time period: 2010 - 2024 (15 years)
  • Geographic coverage: 6 major regions
  • Number of BMW models: 15+ distinct models

Variables

Variable Name Type Description
Model Character BMW model name (e.g., 3 Series, X5, i8)
Year Numeric Year of sale (2010-2024)
Region Character Geographic region (Africa, Asia, Europe, Middle East, North America, South America)
Color Character Vehicle color
Fuel_Type Character Petrol, Diesel, Hybrid, or Electric
Transmission Character Manual or Automatic
Engine_Size_L Numeric Engine size in liters
Mileage_KM Numeric Vehicle mileage in kilometers
Price_USD Numeric Sale price in US dollars
Sales_Volume Numeric Number of units sold
Sales_Classification Categorical Low, Medium, or High sales category

Data Limitations

While this dataset provides valuable insights into BMW's global sales patterns, there are some limitations to consider:

  • Data aggregation at the regional level masks country-specific patterns
  • No information on competitor sales or market share
  • Limited demographic information about buyers
  • Economic indicators (GDP, currency exchange rates) not included
  • Dataset does not distinguish between new and used vehicle sales

Data Usage

This dataset is publicly available on Kaggle under their standard usage terms. We use it for educational and analytical purposes as part of the P8105 Data Science course at Columbia University.
